# README #

Backpackers app is an Web Mobile App developed using Ionic Framework.

### Technologies ###

* Ionic Framework
* Javascript
* AngularJS


### API's ###

##  Google Maps API ##
* Maps screen
* Markers
* Cities Lat and Lng

## Backpackers API ##
* API key
* Authentication
* City Search
* Database


### System Requirements ###
* Node.js [Official Site](http://nodejs.org/)
* Cordova  $ sudo npm install -g cordova
* Ionic $ sudo npm install -g ionic
